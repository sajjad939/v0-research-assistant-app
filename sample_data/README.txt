# Sample Data for Testing

This directory contains example research papers for testing the multi-agent system.

## Contents

1. **paper_1.txt** - Deep Learning for NLP
   - 3 agents analyzed
   - Topics: transformers, attention mechanisms, NLU benchmarks
   - Key metrics: 92.3% accuracy on GLUE

2. **paper_2.txt** - Efficient LLM Training
   - Focus: optimization and distributed training
   - Metrics: 2.8x speedup, 35% memory reduction
   - Infrastructure: 512 TPUs

3. **paper_3.txt** - Multimodal Vision-Language Models
   - Image captioning and zero-shot transfer
   - Datasets: COCO, Flickr30K, ImageNet
   - Performance: State-of-the-art on image captioning

## Quick Start

1. Upload all three files in the Streamlit app
2. Set configuration to 4 agents, 3 rounds
3. Run analysis
4. View synthesized insights in the report

Expected analysis time: ~30 seconds
